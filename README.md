# BilimHub - Образовательная платформа

## 🎓 Описание проекта

BilimHub - это современная образовательная онлайн-платформа для изучения программирования с системой курсов, лекций, тестирования и рейтингом пользователей.

##  Основной функционал

###  Система аутентификации
- Регистрация и вход пользователей
- JWT токены (HTTP-only cookies)
- Хеширование паролей с bcrypt
- Роли: пользователь и администратор

###  Система обучения
- Темы курсов с лекциями
- Markdown контент лекций
- Видео материалы
- Двухуровневая система тестирования
- Автоматическая проверка тестов
- Прогресс-трекинг пользователей
- Последовательная разблокировка лекций

### Тестирование
- Тест 1: основная попытка (проходной балл 70%)
- Тест 2: автоматически открывается при провале теста 1
- Одиночный и множественный выбор ответов
- Автоматический подсчет баллов

### Рейтинговая система
- Накопление баллов за прохождение тестов
- Уровни: Новичок → Средний → Продвинутый → Эксперт
- Таблица лидеров

### Админ-панель
- Управление темами курсов (создание, редактирование, удаление)
- Управление лекциями (создание, редактирование, удаление)
- Статистика платформы
- Доступ только для администраторов

После входа администратор автоматически перенаправляется в админ-панель по адресу `/admin`

### Вход в админку

1. Зайдите на **/login** и войдите под учёткой с ролью администратора.
2. Учётные данные (если выполняли `npm run db:reset` или сиды):
   - **admin@bilimhub.com** — пароль: **admin123**
   - **admin@gmail.com** — пароль: **admin12345**
3. Если после входа пишет «Доступ запрещен» или не перенаправляет в админку — у пользователя в БД нет роли `admin`. Выполните в корне проекта:
   ```bash
   node scripts/setAdminRole.js admin@bilimhub.com
   ```
   (подставьте тот email, под которым входите). Затем снова войдите на /login.



## 📝 Структура БД

### Таблицы
- `users` - пользователи (role: user/admin)
- `themes` - темы курсов
- `lectures` - лекции
- `tests` - тесты (2 на каждую лекцию)
- `questions` - вопросы тестов
- `answers` - варианты ответов
- `user_progress` - прогресс пользователей
- `user_ratings` - рейтинг пользователей

## 🎨 Технологии

### Backend
- Node.js + Express 5.1.0
- PostgreSQL + Sequelize ORM
- JWT + bcrypt
- express-session

### Frontend
- EJS шаблоны
- Vanilla JavaScript
- CSS3 (отдельные файлы стилей)
- Marked.js (Markdown парсер)
